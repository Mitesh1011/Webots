<hr>

<h2>📌 About</h2>
<p>
This repository contains robotics simulations built using <b>Webots</b>, an open-source 
robot simulator used for modelling, programming, and testing autonomous robotic systems 
in a virtual environment.
</p>

<hr>

<h2>🛠️ Project Overview</h2>
<ul>
  <li>Creating a 4-wheeled Differential Drive robot</li>
  <li>Designing and building the robot body</li>
  <li>Adding proximity sensors on the front side</li>
  <li>Writing an obstacle avoidance controller</li>
</ul>

<p><b>Credits:</b> [https://cyberbotics.com/doc/guide/tutorial-6-4-wheels-robot]</p>

<hr>

<h2>⚙️ Features</h2>
<ul>
  <li>Physics-based robot simulation</li>
  <li>Multi-wheel robot configurations</li>
  <li>Sensor integration (distance, camera, etc.)</li>
  <li>Python-based controllers</li>
  <li>Autonomous behaviour implementation</li>
</ul>

<hr>

<h2>🚀 Applications</h2>
<ul>
  <li>Mobile robot navigation</li>
  <li>Obstacle avoidance</li>
  <li>Human-Robot Interaction (HRI)</li>
  <li>Industrial robotics simulation</li>
</ul>
